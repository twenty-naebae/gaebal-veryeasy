## 프로젝트 소개
본 프로젝트는 물류 관리 및 배송 시스템을 위한 MSA 기판 플랫폼입니다. 
대량의 트래픽을 처리할 수 있도록 설계하였으며 사용자는 원활한 주문과 배송 서비스를 이용할 수 있으며, 허브 관리, 배송담당자에게 할당된 배송 메세지를 전달해주는등 편의성을 제공하였습니다. 
확장 가능성을 고려하여 유연하고 확장성 있는 아키텍처 기반으로 작성되었습니다.
